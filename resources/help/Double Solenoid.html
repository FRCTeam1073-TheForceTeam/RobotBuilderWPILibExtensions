Module-based Double Solenoid

What is it?

A solenoid that is plugged the solenoid breakout module. It controls a pneumatic cylinder to create linear motion.

Properties

Forward Module (Solenoid)
The solenoid module on the CRIO that this solenoid is plugged into.
Forward Port (Solenoid)
The port on the solenoid that this solenoid output is plugged into.
Reverse Module (Solenoid)
The solenoid module on the CRIO that this solenoid is plugged into.
Reverse Port (Solenoid)
The port on the solenoid that this solenoid output is plugged into.
