Analog Input

What is it?

An analog input is a variable input that gets converted by an analog to digital converter. Each analog input is read from hardware as a 12-bit number representing -10V to 10V. The raw value or the value converted to volts can be accessed using the analog input object.

Properties

Input Module (Analog)
The module on the CRIO that this potentiometer is plugged into.
Input Port (Analog)
The port on the analog that this potentiometer is plugged into.
