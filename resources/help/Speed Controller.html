Speed Controller

What is it?

A speed controller that uses PWM signals to control the speed of the attached motor. There are three main types of PWM speed controllers available for use in FRC: Victor, Talon and Jaguar.

Properties

Type
The type of speed controller to use.
Output Module (Digital)
The Digital Sidecar on the CRIO that the output is plugged into.
Output Port (PWM)
The PWM port on the digital sidecar that the output is plugged into.
