CAN Jaguar

What is it?

A Jaguar speed controller that uses CAN protocol to control the the attached motor. In addition to controlling speed, Jaguars controlled by CAN can do PID and uses limit switches to control motion.

Properties

CAN ID
The ID of the CAN device. This must match the ID of the device configured with BDC-COMM.
